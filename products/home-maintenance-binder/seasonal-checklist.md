# SEASONAL HOME MAINTENANCE CHECKLIST

### Spring
- [ ] Inspect roof for winter damage
- [ ] Clean gutters and downspouts
- [ ] Service HVAC system (A/C tune-up)
- [ ] Inspect window screens
- [ ] Check exterior drainage

### Summer
- [ ] Test smoke and CO detectors
- [ ] Clean dryer vent
- [ ] Inspect deck/patio for repairs
- [ ] Trim bushes and trees away from house
- [ ] Check irrigation system

### Fall
- [ ] Drain exterior faucets and sprinklers
- [ ] Service furnace/heating system
- [ ] Clean chimney/fireplace
- [ ] Seal cracks in windows and doors
- [ ] Check attic insulation

### Winter
- [ ] Inspect for ice dams
- [ ] Check basement for leaks/dampness
- [ ] Test sump pump
- [ ] Clean kitchen exhaust fan filter
