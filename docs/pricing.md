# Steadfast Property Care - Pricing Structure

## Service Rates
| Service Type | Rate | Notes |
|--------------|------|-------|
| Standard Handyman/Maintenance | $95 / hour | 1 hour minimum |
| Emergency Call-out (After Hours) | $175 fee + hourly | Includes first 30 mins |
| Specialized (Licensed Plumb/Elec) | $125 / hour | |
| Diagnostic/Estimate Fee | $75 | Waived if job is booked |

## Common Flat-Rate Services
- **HVAC Tune-up:** $125
- **Gutter Cleaning:** $150 (up to 2,000 sq ft), $200 (2,000 - 3,500 sq ft)
- **Drain Snaking:** $175 (standard)
- **Lock Rekeying:** $75 service call + $25 per cylinder
- **Toilet Replacement:** $250 (labor only, excludes fixture)
- **Faucet Repair/Replacement:** $150 (labor only)
- **Ceiling Fan Installation:** $185 (standard height)
