# Client Onboarding Checklist: Landlords & Property Managers

This checklist ensures a smooth onboarding process for new professional clients.

## Phase 1: Intake & Assessment
- [ ] Initial Discovery Call / Meeting
    - [ ] Confirm number of properties/units
    - [ ] Identify primary pain points (responsiveness, quality, billing)
    - [ ] Discuss preferred communication channels
- [ ] Property Information Collection
    - [ ] Spreadsheet of all property addresses
    - [ ] Access instructions for each (Lockbox codes, master keys, tenant contact info)
    - [ ] Identification of any known pre-existing issues
- [ ] Site Visit / Initial Inspection
    - [ ] Baseline inspection of major systems (HVAC, Plumbing, Electrical)
    - [ ] Photo documentation of current condition

## Phase 2: Agreements & Admin
- [ ] Select Maintenance Plan
    - [ ] Silver, Gold, or Platinum tier selection
    - [ ] Signed Service Level Agreement (SLA)
- [ ] Account Setup in Team-DB
    - [ ] Client profile created
    - [ ] Billing preferences set (Monthly invoicing, Auto-pay)
- [ ] Emergency Protocol Confirmation
    - [ ] Who is authorized to approve emergency repairs?
    - [ ] Approval threshold amount (e.g., "Approve anything under $500 automatically")

## Phase 3: Launch
- [ ] Welcome Package Sent
    - [ ] Contact card for maintenance requests
    - [ ] Tenant announcement letter (introducing Steadfast Property Care)
- [ ] Schedule first routine maintenance visit
- [ ] Introduce dedicated Account Manager (if Platinum plan)
